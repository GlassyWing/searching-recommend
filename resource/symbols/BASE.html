<static> {Map} BASE.convertJsonToMap(要转换的Json对象)
将Json转换成Map 
 例如： 
         var person=
         { id:"xiaoming", 
         	name:"小明",
         	sex:"男" 
         }
         var person_map=BASE.convertJsonToMap(person); 
         var id=person_map.get("id"); 
         var name=person_map.get("name"); 
         var sex=person_map.get("sex");
         ------------------------------------------------------结果:
         id="xiaoming"; 
         name="小明"; 
         sex="男";
         此处json和map的区别是map取值需要用get方法，设置值需要用set方法
