<static> {void} SERVER.callProcedure(查询id, 传入参数, 回调方法)
调用数据库存储过程 
 例如： 
         SERVER.callProcedure(
         "Procedure.prc_GetNextIntID_Table",
         {
         	vi_column_max:"fhh",vi_tablename:"icbomentry",
         	vi_column:"finterid",vi_columnvalue:data1.finterid,vi_type:"1"
         },
         function(ret1){ // 调用存储过程后使用返回值设置表单的值（带有表单验证） 
         	FORM.setFormValue(
         "detailhh", ret1.get("vo_id") ); 
         } );
         ----------------------------- 结果:
         调用存储过程，sqlid为Procedure.prc_GetNextIntID_Table
         json参数为{vi_column_max:"fhh",vi_tablename:"icbomentry",
			vi_column:"finterid",vi_columnvalue:data1.finterid,vi_type:"1"}
         回调方法为to_setFormValue 调用完后将返回值用于表单赋值操作
