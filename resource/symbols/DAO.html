根据规则递归初始化数据，只接受list,json 
 例如： DAO.initData([
         { 
         	type:data, beanName:"USER",
         	data:{id:13,name:"小明",sex:"男"}},
         	{ beanName:"USER" }
         ])
         数组中的第二个参数会从当前页面中的Form表单中提取数据填充name为USER的bean对象
