<static> {JSON} JSONUTIL.apply(JSON目标对象, JSON合并对象)
合并两个JSON的值，覆盖目标对象中的属性，以最后一次为准 
 例如：  var jsonInfo = { test1 : "测试1" , test2 : "测试2"};
         var jsonInfo1 = {test11 : "测试11" , test12 : "测试12", test1 : "测试1111"}; 
         var resultJsonInfo = JSONUTIL.apply(jsonInfo,jsonInfo1);
         ------------------------------------------------------结果：
         resultJsonInfo = {
         	test1 : "测试1111" , test2 : "测试2" , test11 : "测试11" , test12 : "测试12"
         }
